## About the Project

Chef Twist is a stylish web interface built to represent a food brand or restaurant.  
It focuses on clean design, user experience, and visually appealing layout.

The project demonstrates front-end development skills using core web technologies.

---

## ✨ Features

- 🍔 Clean and modern UI design  
- 📱 Fully responsive (mobile & desktop)  
- 📖 Sections for menu, about, and services  
- 🎯 Smooth navigation and layout  
- 🎨 Styled components with CSS
